Description
----------------

Excerpt module adds an 'excerpt' field to node forms, allowing you to
enter a separate excerpt/summary/teaser for a node. It i an alternatve to the 
"teaser splitter" feature of Drupal6 core.

Configuration
-----------------

(optional) If you only want separate excerpts on some content types, then disable 
the Manual teaser choice on the edit form for that content type.
